Output Pane

This pane is always available. It displays the results of compilation, composition and analysis. LTSA automatically switches to this pane when new output is produced. If an error is discovered during compilation, the error message is displayed in this pane. To discover where the error is in the source switch back to the Edit pane and the cursor will be positioned at the error and the character at the start of erroneous token highlighted.

The LTSA does not switch to the output pane if a syntax  error is discovered for a parse operation (without compilation). The usual symptom is that the target or run selection is not updated. Switch to the output pane to discover the cause of the error.
